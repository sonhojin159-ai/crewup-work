# Model Routing Rules

Hermes 20달러 플랜을 관제탑으로 두고 각 모델/CLI를 역할별로 사용하는 기준입니다.

## Hermes
- 전체 관제탑
- 작업 지시서 작성
- 결과 보고서 요약
- Obsidian 저장
- 반복 가능한 절차/체크리스트 관리

## Claude / Claude Code
- 긴 문서 이해
- 복잡한 기획/구조 설계
- 신규 메인 구현 지시 대상
- 깊은 UX/제품 판단

## Codex
- 기존 코드 수정
- 버그 패치
- UI 미세 조정
- 작은 기능 변경

## Grok
- X/실시간 트렌드
- 여론/화제성 조사
- 빠른 시장 감각 확인

## OpenAI / ChatGPT 계열
- 범용 정리
- 아이디어 확장
- 구조화/요약
- 멀티모달/이미지 관련 작업이 필요할 때

## Anti-Gravity
- 최신 웹/API 조사
- 시안/prototype/review
- 구현 완료 후 보안/버그/UX 검수
