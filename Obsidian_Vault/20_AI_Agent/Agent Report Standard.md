# Agent Report Standard

각 모델/에이전트가 작업 후 제출해야 하는 표준 보고서 형식입니다.

```md
# [Agent Report]

- 작업 상태: 성공 / 실패 / 부분 완료
- 담당 에이전트/모델:
- 작업 범위:
- 생성/수정된 파일:
- 핵심 요약:
- 발견한 문제:
- 프로젝트에 적용 가능한 점:
- 추천 다음 작업:
- 참고 링크:
- 에러/주의사항:
```

## 저장 원칙
- 코드 전문이 아니라 판단에 필요한 팩트 요약 중심
- 일회성 로그보다 재사용 가능한 지식 중심
- 다음 작업으로 이어질 내용은 `Action Items`에 별도 정리
