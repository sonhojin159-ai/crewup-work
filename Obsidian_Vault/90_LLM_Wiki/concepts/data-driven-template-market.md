# Data Driven Template Market

`양식 하나당 판매`하되, 핵심 자산은 HTML 파일 자체가 아니라 `사용자 입력 데이터 → 업종별 스키마 → 여러 양식 렌더링` 구조다.

## 의미
기존 상세페이지/랜딩페이지 외주와 달리, 같은 입력값으로 룩북형·포스터형·인스타형·예약형·사례형을 바꿔볼 수 있어야 한다. 이 구조는 [[mobile-promo-page-builder]]의 상품성을 만든다.

## 현재 입력 데이터 축
- 매장명
- 업종/주소/링크/연락처
- 메인 제목/설명문/CTA
- 판매 품목명/가격/설명
- 이벤트명/기간/혜택
- 대표 사진/갤러리 사진
- 예약 가능 시간

## 판매 구조
- 크몽: 19,000원~29,000원 기본형으로 시작 가능
- 자체 웹사이트: 자동화 수준이 높아지면 9,900원 상품 가능
- 업셀: 문구 정리, 사진 정리, 추가 양식 비교, 링크 발급, 수정 횟수 추가
