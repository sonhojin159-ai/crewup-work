# Low Token Builder Flow

AI 토큰 비용을 낮추기 위해, 대부분의 결과물은 자유 생성이 아니라 선택지 기반 UX와 규칙 기반 렌더링으로 만든다.

## 원칙
- 사용자는 가능한 한 카드, 드롭다운, 토글, 프리셋을 선택한다.
- 1차 생성은 AI가 아니라 [[data-driven-template-market]]의 템플릿 엔진이 처리한다.
- AI는 꼭 필요한 순간에 짧게 호출한다.
- 자연어 수정 요청은 유료 전환 직전 또는 제한된 횟수로 제공한다.

## 권장 선택 흐름
1. 업종 선택
2. 양식 선택
3. 분위기 선택: 고급스러운, 귀여운, 깔끔한, 감성적인, 강한 이벤트형
4. 목적 선택: 예약 문의, 가격표 안내, 이벤트 홍보, 후기/사례 강조
5. 섹션 순서 선택
6. 문구 후보 선택
7. 제한된 AI 수정 요청
