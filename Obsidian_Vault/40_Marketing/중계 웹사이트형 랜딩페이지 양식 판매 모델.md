# 중계 웹사이트형 랜딩페이지 양식 판매 모델

작성일: 2026-06-09

## 핵심 방향 전환

사용자는 수정 가능한 랜딩페이지를 완성품으로 납품하는 방식이 아니라, `웹페이지로 중계 웹사이트를 만들고 양식 하나당 저렴하게 판매하는 구조`를 제안했다.

핵심 문장:

> 웹페이지로 중계 웹사이트를 만드는 거야. 양식 하나 당 돈을 받고 팔고, 대신 완성품을 주는 게 아니니까 가격은 저렴해야 해.

## 제품 정의

이 상품은 기존의 “랜딩페이지 제작 대행”이 아니라 다음에 가깝다.

```text
소상공인용 모바일 안내페이지 양식 마켓
```

또는:

```text
엑셀/입력폼으로 내용을 넣으면 여러 모바일 홍보페이지 양식에 미리보기로 반영되는 웹사이트
```

## 핵심 구조

사용자는 중계 웹사이트에서 다음 흐름을 경험한다.

```text
1. 업종 또는 목적 선택
2. 양식 타입 선택
3. 엑셀 또는 웹 입력폼으로 정보 입력
4. 미리보기 확인
5. 원하는 양식 하나 구매
6. HTML 파일, 링크, 또는 배포 옵션 제공
```

## 중요한 차이

### 기존 완성품 납품형

```text
고객 요청 수집
→ 제작자가 직접 디자인/수정
→ 완성품 납품
→ 가격 높음
→ 수정 요청 많음
```

### 중계 웹사이트/양식 판매형

```text
고객이 직접 입력
→ 양식에 자동 반영
→ 미리보기 확인
→ 양식 단위 결제
→ 가격 낮음
→ 제작자 개입 최소화
```

## 가격 방향

완성품 납품이 아니므로 가격은 저렴해야 한다.

가능한 가격대:

1. 단일 양식 구매
   - 9,900원 ~ 19,900원
   - HTML 파일 또는 공유 링크 제공

2. 양식 3개 묶음
   - 29,000원 ~ 49,000원
   - 같은 입력 데이터로 3가지 양식 제공

3. 업종별 스타터팩
   - 49,000원 ~ 99,000원
   - 네일샵/필라테스/에어컨청소 등 업종별 양식 여러 개

4. 월 구독/관리형
   - 월 9,900원 ~ 29,000원
   - 호스팅, 링크 유지, 월 1회 수정, 이벤트 업데이트 포함

## 핵심 판매 논리

고객에게는 이렇게 보여줘야 한다.

```text
랜딩페이지 제작비 20~50만원이 부담된다면,
먼저 9,900원짜리 모바일 안내페이지 양식으로 시작하세요.
```

또는:

```text
사장님이 엑셀/입력폼에 가격표와 이벤트만 넣으면,
인스타 프로필에 걸 수 있는 모바일 안내페이지가 바로 만들어집니다.
```

## 중계 웹사이트에 필요한 기능

MVP 필수 기능:

1. 양식 카탈로그
   - 룩북형
   - 포스터형
   - 인스타 피드형
   - 견적 문의형
   - 예약 캘린더형
   - 전후 사례형
   - 가격표형

2. 데이터 입력 방식
   - 엑셀 업로드
   - 또는 웹 입력폼
   - 최소 항목: 매장명, 가격표, 이벤트, 이미지, CTA 링크

3. 실시간 미리보기
   - 모바일 390px 프레임
   - 양식 변경 시 같은 데이터로 디자인 전환

4. 결제 전/후 구분
   - 결제 전: 워터마크 또는 일부 제한
   - 결제 후: HTML 다운로드 또는 공유 링크 생성

5. 구매 단위
   - 양식 1개 단위 판매
   - 같은 입력 데이터로 다른 양식 추가 구매 가능

## 완성품을 주지 않는다는 의미

여기서 “완성품을 주지 않는다”는 말은 고객별 맞춤 제작과 수정 대행을 기본으로 제공하지 않는다는 뜻이다.

제공하는 것:

```text
정해진 양식
입력값 반영
미리보기
HTML/링크 생성
```

기본 제공하지 않는 것:

```text
개별 디자인 커스터마이징
카피라이팅 대행
사진 보정
무제한 수정
업종별 맞춤 전략 설계
```

따라서 가격을 낮출 수 있다.

## 업셀 구조

저렴한 양식 판매에서 끝내지 않고 다음으로 업셀할 수 있다.

1. 기본 양식 구매
   - 9,900원~19,900원

2. 맞춤 세팅 대행
   - 49,000원~99,000원
   - 고객의 엑셀/사진을 대신 정리해서 예쁘게 세팅

3. 월관리
   - 월 19,900원~49,000원
   - 이벤트/가격표/예약시간 월 1~2회 수정

4. 맞춤 제작
   - 150,000원~300,000원
   - 양식으로 부족한 고객에게 별도 제작

## 중요한 제품 판단

이 모델의 핵심은 `대행 전에 저가 양식 상품으로 진입장벽을 낮추는 것`이다.

단순 대행은 레드오션이지만, 다음 조합은 차별점이 있다.

```text
양식 마켓 + 엑셀 입력 + 모바일 미리보기 + 저가 구매 + 필요 시 대행 업셀
```

## MVP 제안

처음부터 복잡한 관리자/결제 시스템을 만들 필요는 없다.

1차 MVP:

```text
네일샵 6개 양식
엑셀 입력 템플릿
브라우저에서 JSON 데이터로 미리보기
HTML 다운로드 버튼
가짜 결제 버튼 또는 문의 버튼
```

2차 MVP:

```text
업종 3개 추가
웹 입력폼 추가
양식별 가격 표시
구매 문의 카카오톡 연결
```

3차 MVP:

```text
실제 결제
자동 HTML 생성
호스팅 링크 발급
구글시트 연동
```

## 현재 가장 적합한 상품명 후보

- 모바일 안내페이지 양식 마켓
- 사장님용 홍보페이지 템플릿 마켓
- 엑셀 수정형 모바일 페이지 양식
- 인스타 프로필용 모바일 안내페이지 템플릿
- 소상공인 홍보페이지 양식 상점
