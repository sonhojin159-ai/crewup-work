# 소상공인 문의 전환 랜딩페이지 기획안

작성일: 2026-06-08  
관련 노트: [[100-150만원 월수익 전략]], [[Marketing Knowledge Base]]

## 1. 한 줄 정의

소상공인의 인스타, 블로그, 네이버 플레이스 방문자를 `전화/카카오톡/신청폼 문의`로 연결하는 모바일 1페이지 홍보페이지.

핵심은 예쁜 홈페이지 제작이 아니라 `문의/예약/상담 전환 시스템`이다.

## 2. 판매 포지셔닝

### 상품명 후보

- 대표님 가게 문의 전환 페이지
- 소상공인 문의형 1페이지 홍보페이지
- 인스타·블로그 방문자를 카톡 문의로 연결하는 홍보페이지
- 가격/서비스/후기/문의 버튼을 한 페이지에 정리하는 모바일 홍보페이지

### 고객에게 말할 문장

> 인스타나 블로그를 보고 들어온 손님이 가격, 서비스, 후기, 위치를 빠르게 확인하고 바로 카카오톡이나 전화로 문의할 수 있게 1페이지로 정리해드립니다.

### 팔지 말아야 할 표현

- 랜딩페이지 만들어드립니다
- AI로 예쁜 페이지 만들어드립니다
- 저렴한 홈페이지 제작

### 팔아야 할 표현

- 문의로 이어지는 구조를 만들어드립니다
- 가격/서비스/후기/문의 버튼을 모바일에서 바로 보이게 정리합니다
- 대표님은 가격, 공지, 사진, 문구만 쉽게 바꾸면 됩니다

## 3. 1차 타깃 업종

초기에는 `상담/예약/견적`이 필요한 업종이 좋다.

우선순위:

1. PT샵 / 필라테스 / 체형교정
2. 피부관리실 / 네일샵 / 미용실
3. 공방 / 클래스 / 원데이클래스
4. 청소 / 수리 / 인테리어 / 출장 서비스
5. 학원 / 과외 / 상담형 교육

선정 기준:

- 인스타나 블로그는 있지만 링크를 눌렀을 때 정리된 문의 페이지가 없다
- 가격, 서비스, 예약 방법, FAQ가 흩어져 있다
- 고객이 문의 전에 반복해서 묻는 질문이 많다
- 공지, 가격, 사진, 일정이 자주 바뀐다
- 월관리 전환 가능성이 있다

## 4. 랜딩페이지 목적

1. 첫 방문자가 5초 안에 `무슨 서비스인지` 이해한다.
2. 가격/서비스/후기/위치/문의 방법을 한 번에 확인한다.
3. 고객이 문의 전에 가질 불안을 줄인다.
4. 카카오톡/전화/폼 중 하나로 바로 전환시킨다.
5. 제작 후 대표가 가격, 공지, 사진, 후기, CTA 링크를 직접 바꿀 수 있게 한다.

## 5. 전환 원칙

### Message Match

광고, DM, 인스타 프로필, 블로그에서 약속한 문장이 랜딩페이지 첫 화면에 그대로 보여야 한다.

예:

- DM 문구: `가격, 위치, 예약 방법을 한 페이지에 정리해드립니다`
- 히어로 문구: `가격, 서비스, 후기, 예약 문의를 한 페이지에서 바로 확인하세요`

### CTA 단순화

주 CTA는 하나로 정한다.

- 예약형 업종: `카카오톡으로 예약 문의하기`
- 견적형 업종: `사진 보내고 견적 문의하기`
- 상담형 업종: `내 상황 상담받기`
- 클래스형 업종: `수업 일정 문의하기`

보조 CTA는 전화하기 또는 폼 신청 정도만 둔다.

### 모바일 우선

모든 섹션은 모바일에서 먼저 설계한다.

- 첫 화면에 CTA가 반드시 보여야 한다
- 버튼 높이는 최소 48px 이상
- 하단 스티키 CTA를 둔다
- 섹션 제목은 짧고 명확하게 쓴다
- 긴 설명보다 카드형 정보 구조를 쓴다

## 6. 페이지 구조

### 1. 히어로 섹션

목적: 첫 화면에서 업종, 핵심 결과, 문의 버튼을 즉시 보여준다.

구성:

- 브랜드/매장명
- 업종 또는 핵심 서비스
- 메인 카피
- 서브 카피
- 대표 이미지
- CTA 버튼 2개
- 보조 정보: 지역, 운영시간, 첫 상담 가능 여부

예시:

> 강남 1:1 체형교정 PT  
> 통증과 자세 고민을 상담부터 운동 루틴까지 맞춤 관리합니다.  
> 첫 상담 예약 가능 · 카카오톡 문의 1분 연결

CTA:

- 카카오톡으로 상담하기
- 전화로 바로 문의하기

### 2. 고객 문제 제기 섹션

목적: 방문자가 가진 불안과 판단 장벽을 짚는다.

예시 문구:

- 가격이 어떻게 되는지 모르겠어요
- 나에게 맞는 서비스인지 판단하기 어려워요
- 상담 전에 실제 후기나 사례를 보고 싶어요
- 예약 방법과 준비물이 헷갈려요

섹션 제목 후보:

- 문의 전 이런 점이 궁금하셨나요?
- 처음 방문 전 가장 많이 묻는 질문입니다
- 상담 전에 확인하면 좋은 정보만 정리했습니다

### 3. 대표 서비스/상품 섹션

목적: 대표 상품 3개를 카드형으로 보여준다.

카드 필드:

- 서비스명
- 한 줄 설명
- 가격 또는 상담 안내
- 소요 시간
- 추천 대상
- 포함 사항

예시:

```text
기본 상담
30분 / 30,000원
처음 방문하시는 분께 추천
현재 상태 확인 + 맞춤 방향 안내
```

가격 공개가 어려운 경우:

```text
상담 후 견적
현장 상태, 요청 범위, 일정에 따라 안내드립니다.
카카오톡으로 사진을 보내주시면 대략적인 비용을 먼저 안내드립니다.
```

### 4. 추천 패키지 섹션

목적: 고객이 무엇을 선택해야 하는지 쉽게 만든다.

구성:

- 기본형
- 추천형
- 정기/관리형

표시 원칙:

- 가장 팔고 싶은 상품에 `추천` 배지
- 가격이 어렵다면 `상담 후 안내`와 기준 범위를 함께 사용
- 선택 기준을 명확히 표시

예시:

```text
처음 방문 상담
처음 이용하시는 분께 추천
30분 / 30,000원

대표 프로그램 추천
가장 많이 선택하는 구성
4회 / 220,000원

정기 관리
꾸준한 변화가 필요한 분께 추천
월 4회 / 상담 후 안내
```

### 5. 후기/신뢰 섹션

목적: 문의 전 불신을 줄인다.

후기가 있을 때:

- 실제 고객 후기 2~3개
- 전후 사진
- 카카오톡 후기 캡처
- 네이버 플레이스 리뷰 링크

후기가 없을 때 대체 요소:

- 대표자 얼굴 또는 작업 사진
- 작업 과정 사진
- 자격증/경력
- 사용하는 도구
- 상담 전후 체크리스트
- 샘플 결과물

후기 예시:

> 가격과 진행 방식이 한눈에 보여서 문의하기 편했어요.

### 6. 진행 방식 섹션

목적: 문의 이후 흐름을 예측 가능하게 만든다.

예시:

1. 카카오톡 문의
2. 현재 상황/희망 일정 확인
3. 가격 또는 가능 여부 안내
4. 예약 확정
5. 방문/진행

카카오톡 문의 직후 자동 안내 문구:

```text
안녕하세요. 빠른 안내를 위해 아래 내용을 함께 보내주세요.

1. 원하시는 서비스 또는 클래스
2. 희망 날짜와 시간
3. 궁금한 점
4. 참고 사진이 있다면 첨부
```

### 7. FAQ 섹션

목적: 반복 문의와 이탈을 줄인다.

필수 FAQ:

- 가격은 어떻게 되나요?
- 예약은 어떻게 하나요?
- 소요 시간은 얼마나 걸리나요?
- 처음 방문해도 괜찮나요?
- 변경/취소는 어떻게 하나요?
- 주차나 위치는 어떻게 되나요?

FAQ는 단순 안내가 아니라 전환 장치다. 문의 전에 막히는 질문을 먼저 제거해야 한다.

### 8. 위치/운영 정보 섹션

구성:

- 주소
- 영업시간
- 휴무일
- 주차 가능 여부
- 대중교통 안내
- 네이버 지도 링크
- 인스타/블로그 링크

### 9. 마지막 CTA 섹션

목적: 스크롤 끝에서 다시 행동을 유도한다.

예시:

> 상담 전 궁금한 점이 있다면 카카오톡으로 편하게 보내주세요.  
> 사진이나 희망 날짜를 함께 보내주시면 더 빠르게 안내드립니다.

CTA:

- 카카오톡으로 문의하기
- 전화로 바로 문의하기
- 신청폼 작성하기

## 7. 스티키 CTA 설계

모바일 하단에 고정 버튼을 둔다.

버튼 문구 후보:

- 카카오톡으로 1분 상담하기
- 사진 보내고 견적 문의하기
- 이번 주 예약 가능 시간 보기
- 수업 일정 문의하기
- 가격 상담하기

설계 규칙:

- 높이 48~56px 이상
- 배경과 명확히 분리
- 한 화면에 CTA는 하나만 강조
- 3~5초 주기의 과한 애니메이션은 피하고, 필요하면 아주 약한 펄스만 사용

## 8. 고객용 직접수정 필드

대표가 직접 바꿀 수 있는 영역은 제한한다.

수정 가능 필드:

- 매장명
- 대표 이미지
- 메인 문장
- 보조 문장
- 서비스/가격
- 공지/이벤트
- 후기
- FAQ
- 카카오톡 링크
- 전화번호
- 네이버 예약 링크
- 인스타그램 링크
- 영업시간
- 휴무일

수정 불가 영역:

- 전체 레이아웃
- 섹션 순서
- 디자인 시스템
- 버튼 스타일
- 전환 흐름

원칙:

> 사장님이 마음대로 꾸미는 웹 편집기가 아니라, 정해진 구조 안에서 장사 정보를 바꾸는 입력표여야 한다.

## 9. 제작 전 자료 요청 체크리스트

필수 자료:

- 매장명
- 주소
- 영업시간
- 대표 서비스 3개
- 가격표 또는 기준 가격
- 카카오톡 문의 링크
- 전화번호
- 대표 사진 3장 이상

있으면 좋은 자료:

- 후기
- 전후 사진
- 공간 사진
- 시술/작업 과정 사진
- 기존 인스타그램 링크
- 블로그 링크
- 네이버 플레이스 링크

자료가 없을 때 대체 방식:

- 가격표 없음: 기준 가격 또는 `상담 후 안내` 문구 사용
- 후기 없음: 자주 듣는 고객 반응을 짧은 문장으로 정리
- 사진 부족: 보유 사진으로 우선 오픈 후 1개월 내 교체
- FAQ 없음: 상담 질문지에서 반복 문의를 뽑아 구성

## 10. 첫 샘플 업종 제안

첫 포트폴리오 샘플은 `PT/체형교정` 또는 `피부관리실`이 좋다.

이유:

- 모바일 문의 전환이 중요하다
- 가격/상담/후기/전후 사진 구조가 명확하다
- 카카오톡 상담으로 이어지기 쉽다
- 월관리 명분이 있다

추천 첫 샘플:

> `1:1 체형교정 PT 문의 전환 페이지`

핵심 섹션:

- 통증/자세 고민 히어로
- 처음 방문 전 고민 4개
- 대표 프로그램 3개
- 상담 진행 방식
- 후기/전후 사례
- FAQ
- 카카오톡 상담 CTA

## 11. 가격 패키지 초안

### 스타터: 29만원

- 모바일 홍보페이지 1개
- 히어로/서비스/가격/FAQ/문의 섹션
- 카카오톡 또는 전화 버튼 연결
- 기본 카피 정리
- 1개월 무료관리 포함

### 베이직: 49만원

- 스타터 포함
- 후기/사진/진행 방식 섹션 추가
- 직접수정 가능 필드 세팅
- 네이버/인스타/블로그 링크 정리
- 1개월 무료관리 포함

### 관리형: 월 10만~15만원

- 월 2회 수정
- 공지/이벤트 교체
- 가격/FAQ 보완
- 문의 버튼 점검
- 월 1회 간단 개선 제안

범위 제한 문구:

> 월관리는 홈페이지를 매번 새로 디자인하는 서비스가 아니라, 고객이 보는 공지, 가격, FAQ, 사진, 문의 문구를 최신 상태로 정리하는 서비스입니다. 디자인 변경이나 새 섹션 제작은 별도 작업으로 안내드립니다.

## 12. 첫 고객 확보 실험

목표:

- 후보 30곳 리스트업
- 무료 진단 5곳 제안
- 샘플 구조 2개 발송
- 체험가 유료 고객 1명 확보

진행:

1. 인스타/네이버 플레이스에서 후보 30곳 수집
2. 가격/예약/FAQ/문의 버튼이 부족한 곳 표시
3. 무료 진단 DM 발송
4. 관심 보인 곳에 1페이지 개선 예시 전달
5. 체험가 29만원 제안

DM 예시:

```text
안녕하세요.
현재 운영 중이신 인스타/네이버 정보를 봤는데, 서비스 설명은 잘 되어 있는데 고객이 가격, 예약 방법, 문의 버튼을 한 번에 확인하기는 조금 어려워 보여 연락드렸습니다.
```

## 13. 성과 측정 지표

초기에는 복잡한 광고 지표보다 아래만 본다.

- 페이지 방문 수
- 카카오톡 버튼 클릭 수
- 전화 버튼 클릭 수
- 폼 제출 수
- 문의 후 실제 예약/상담 수
- 가장 많이 눌린 CTA 문구
- FAQ 추가 후 반복 문의 감소 여부

기준:

- 방문 대비 문의 클릭률 5% 미만: 히어로/오퍼/CTA 수정
- 방문 대비 문의 클릭률 10% 이상: 기본 구조 유효
- 문의는 있는데 예약이 낮음: 가격/신뢰/상담 스크립트 보강
- 방문 자체가 낮음: 인스타/네이버/블로그 유입 경로 보강

## 14. MVP 제작 순서

1. 첫 샘플 업종 결정: PT/체형교정 추천
2. 모바일 와이어프레임 작성

4. 대표 서비스 카드 3개 작성

8. 실제 샘플 페이지 제작
9. 후보 고객 30곳에 무료 진단 DM 발송
10. 반응을 바탕으로 카피/섹션 수정

## 15. 바로 만들 샘플의 핵심 카피

헤드카피 후보:

1. 통증과 자세 고민, 상담부터 운동 루틴까지 맞춤 관리합니다
2. 처음 방문 전 가격·과정·후기를 한 번에 확인하세요
3. 내 몸 상태에 맞는 1:1 체형교정 상담을 카카오톡으로 예약하세요

서브카피 후보:

1. 강남 1:1 체형교정 PT · 첫 상담 예약 가능 · 카카오톡 1분 문의
2. 가격, 소요시간, 진행 방식, 자주 묻는 질문을 한 페이지에 정리했습니다
3. 사진이나 현재 고민을 보내주시면 가능한 프로그램을 먼저 안내드립니다

CTA 후보:

- 카카오톡으로 첫 상담 문의하기
- 이번 주 예약 가능 시간 보기
- 내 상태에 맞는 프로그램 물어보기

## 16. 결론

첫 랜딩페이지는 범용 홈페이지가 아니라 `문의 전환용 모바일 1페이지`로 만든다.

가장 중요한 차별점은 다음 3가지다.

1. 사용자가 만들어주고, 대표는 제한된 정보만 직접 수정한다.
2. 페이지 목적은 소개가 아니라 카카오톡/전화/폼 문의 전환이다.
3. 제작 후 1개월 무료관리로 월관리 전환을 유도한다.

다음 실행은 `PT/체형교정 샘플 랜딩페이지 와이어프레임`을 실제 화면 단위로 만드는 것이다.
