# 모바일 안내페이지 타이포그래피 수정 기록

작성일: 2026-06-09

## 사용자 피드백

사용자는 네일샵 양식별 샘플 6종에 대해 전체 페이지 글자체를 더 동글동글하게 만들 수 있는지 요청했다. 이후 특정 폰트 선택은 AI가 판단하되, 상하좌우 여백·자간·행간을 최적화해서 글자를 읽을 때 불편함이 없기를 원했다.

## 적용한 디자인 판단

무작정 귀여운 둥근 폰트로 가면 페이지가 유치하거나 저가형으로 보일 수 있다. 네일샵/뷰티 모바일 안내페이지에는 다음 균형이 중요하다.

- 너무 각진 시스템 폰트: 기본 템플릿 느낌이 강함
- 너무 둥근 귀여운 폰트: 저가 이벤트 페이지처럼 보일 수 있음
- 적합한 방향: 약간 둥글고 현대적인 산세리프, 모바일 가독성 우선

## 적용 폰트

`SUIT` 웹폰트를 6개 HTML 전체에 적용했다.

폴백 순서:

```css
font-family: 'SUIT', 'Pretendard', 'Noto Sans KR', -apple-system, BlinkMacSystemFont, 'Segoe UI', Arial, sans-serif;
```

선택 이유:

- 한글 모바일 가독성이 좋다.
- Pretendard보다 조금 더 부드럽고 단정한 인상을 준다.
- 너무 귀엽거나 장식적이지 않아 네일샵/뷰티 업종에 무난하다.
- 고객용 실제 페이지에서도 과하지 않다.

## 공통 타이포그래피 조정

6개 HTML 전체에 다음 기준을 적용했다.

```css
letter-spacing: -.012em;
line-height: 1.48;
text-rendering: optimizeLegibility;
word-break: keep-all;
```

의도:

- 자간을 과하게 조이지 않고 약간만 정돈한다.
- 본문 행간은 모바일에서 답답하지 않게 1.48 기준으로 둔다.
- 한글 단어가 어색하게 끊기지 않도록 `word-break: keep-all` 적용.
- 가독성 우선으로 렌더링 옵션 추가.

## 추가 수정

- 인스타형/견적형/포스터형의 가로 스크롤바가 시각적으로 거슬려 숨김 처리.
- 디자인 견적형의 추천 견적 카드에서 원형 이미지가 설명 텍스트와 겹쳐 보일 수 있어 이미지 크기와 위치를 조정하고 설명 문단 폭을 제한.
- 전후 사례형의 히어로 제목이 너무 조밀하게 보일 수 있어 제목 크기/자간/행간을 완화.

## 향후 디자인 원칙

모바일 안내페이지 상품은 단순히 구조만 만드는 것이 아니라, 업종별로 다음을 함께 최적화해야 한다.

1. 양식 타입
2. 이미지 사용 방식
3. 폰트 성격
4. 자간/행간
5. 카드 내부 여백
6. 하단 CTA 여백
7. 모바일 스크롤 피로도

특히 네일샵/뷰티 업종은 `부드러운 산세리프 + 넉넉한 카드 여백 + 이미지 중심 구조`가 기본값으로 적합하다.
