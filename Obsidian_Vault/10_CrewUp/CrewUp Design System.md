# CrewUp Design System

CrewUp UI/UX 리서치와 실제 구현에서 반복적으로 사용할 디자인 기준입니다.

## 브랜드 방향
- 성장형 크루 플랫폼
- AI 목적의 커뮤니티/워크스페이스
- 너무 강의 플랫폼처럼 보이지 않게 함
- 크루장이 선생님이라기보다 함께 세팅하고 공유하는 리더 느낌

## UI 우선순위
1. 크루 발견이 쉬운 랜딩
2. 인기/추천 크루를 한눈에 보여주는 카드 UI
3. 상세보기/참여신청 CTA의 일관성
4. PC/모바일 동일 데이터 연동
5. 크루 관리 페이지에서 내 크루 전환 가능

## 카드 UI 원칙
- 크루 제목/설명/태그/멤버수/상태/CTA가 명확해야 함
- 상세보기와 참여신청 버튼은 같은 위치에 고정되는 느낌이어야 함
- 모바일에서는 카드 간 간격과 버튼 터치 영역을 우선
