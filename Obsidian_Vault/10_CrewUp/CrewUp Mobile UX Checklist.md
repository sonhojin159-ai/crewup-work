# CrewUp Mobile UX Checklist

PC 웹 작업 후 모바일 연동/반응형을 검수할 때 사용하는 체크리스트입니다.

## 데이터 연동
- [ ] PC에서 보이는 크루 목록이 모바일에서도 동일하게 보이는가
- [ ] Supabase/라이브 데이터 기준으로 PC/모바일이 같은 데이터를 사용하는가
- [ ] 하드코딩된 가짜 크루 카드가 남아 있지 않은가
- [ ] 로그인 상태/권한에 따라 내 크루 목록이 모바일에서도 동일하게 보이는가

## 랜딩 페이지
- [ ] 크루 목록을 한눈에 볼 수 있는가
- [ ] 크루 상세보기 버튼 위치가 카드마다 일관적인가
- [ ] 참여신청 버튼 위치가 카드마다 일관적인가
- [ ] CTA가 모바일에서 접히거나 밀리지 않는가
- [ ] 탭/섹션 이동이 모바일에서 정상 작동하는가

## 크루 관리 페이지
- [ ] 랜딩 페이지로 돌아가는 버튼이 있는가
- [ ] 가입한 크루와 크루장으로 운영 중인 크루가 모두 보이는가
- [ ] 여러 크루 사이를 전환할 수 있는가
- [ ] 선택한 크루 기준으로 관리 데이터가 바뀌는가

## 레이아웃
- [ ] 360px~430px 폭에서 깨지지 않는가
- [ ] 버튼 터치 영역이 충분한가
- [ ] 카드 높이가 달라도 주요 CTA 위치가 어색하지 않은가
- [ ] 가로 스크롤이 생기지 않는가
