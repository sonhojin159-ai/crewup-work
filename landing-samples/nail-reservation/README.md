# 온네일 성수 모바일 예약 안내 페이지 샘플

## 선택 업종

네일샵 / 1인 네일샵 / 인스타 프로필용 예약 안내 페이지

## 이번 버전의 방향

이전 PT 샘플의 문제였던 `AI가 만든 랜딩페이지 데모 느낌`을 줄이고, 실제 소상공인이 인스타 프로필 링크에 걸 수 있는 모바일 안내 페이지처럼 설계했다.

## 핵심 차이

- PC 랜딩페이지가 아니라 모바일 390~430px 폭 기준.
- 추상적인 히어로 그래픽보다 가격표, 이달의 아트, 예약 안내, FAQ 중심.
- 고객이 문의 전에 궁금해하는 반복 질문을 먼저 해결.
- 하단 고정 CTA로 카카오톡 예약 문의를 계속 노출.
- 대표가 직접 바꿀 수 있는 필드를 명확히 제시: 이달의 아트, 공지, 가격표, CTA 링크.

## 다음 개선

1. 실제 매장명/가격/사진을 넣으면 판매 샘플로 전환 가능.
2. 관리자 수정폼을 별도 페이지로 만들어 `이달의 아트`, `가격표`, `공지`를 바꿔보는 데모를 만들 수 있음.
3. 크몽 상품 상세페이지에는 `인스타 프로필 링크에 걸 수 있는 모바일 예약 안내 페이지`로 표현하는 것이 좋음.
