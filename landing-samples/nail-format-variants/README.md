# 네일샵 같은 주제 / 다른 양식 6종

같은 `온네일 성수` 주제로 양식 자체가 다르게 보이도록 만든 모바일 안내페이지 샘플.

## 1. lookbook-gallery

- 유형: 룩북/매거진형
- 특징: 큰 풀스크린 사진, 감성 카피, 큰 아트 카드, 프리미엄 분위기
- 잘 맞는 경우: 디자인 결과물이 중요한 네일샵, 피부관리실, 헤어살롱, 공방
- 약점: 정보 탐색 속도는 일반형보다 느릴 수 있음

## 2. bold-poster-event

- 유형: 포스터/이벤트형
- 특징: 강한 노란 배경, 큰 타이포, 혜택/행사 중심, 눈에 띄는 카드
- 잘 맞는 경우: 신규 오픈, 첫 방문 혜택, 월간 프로모션, 팝업/이벤트
- 약점: 프리미엄/차분한 매장에는 과해 보일 수 있음

## 3. instagram-story-feed

- 유형: 인스타 스토리/피드형
- 특징: 스토리 썸네일, 피드형 사진 카드, 예약 가능 캘린더, 가격표
- 잘 맞는 경우: 인스타 운영이 활발한 1인샵, 사진이 자주 바뀌는 매장
- 약점: 인스타 감성이 안 맞는 업종에는 덜 어울림

## 4. design-finder-quote

- 유형: 디자인 선택/견적 문의형
- 특징: 고객이 분위기, 손톱 상태, 희망 시간을 체크하고 예상 견적을 본 뒤 카톡으로 문의
- 잘 맞는 경우: 디자인별 가격 차이가 크고 상담 문의가 많은 네일샵/미용/청소/시공 업종
- 약점: 실제 계산 로직이나 관리자 데이터가 없으면 정적 샘플에 머문다.

## 5. reservation-calendar

- 유형: 예약 캘린더형
- 특징: 이번 주 가능한 시간표와 예약 슬롯을 먼저 보여줌
- 잘 맞는 경우: 1인샵, 필라테스, PT, 클래스, 병원/클리닉처럼 시간 선택이 문의 핵심인 업종
- 약점: 예약 현황이 실시간으로 업데이트되지 않으면 신뢰가 떨어진다.

## 6. before-after-case

- 유형: 전후 사례/케이스 스터디형
- 특징: Before/After, 과정, 소요시간, 예상금액, 유사 사례 중심
- 잘 맞는 경우: 네일 케어, 피부관리, 청소, 수리, 시공, 다이어트/PT처럼 결과 증명이 중요한 업종
- 약점: 실제 전후 사진이 없으면 설득력이 약해진다.

## 의도

이전 샘플들이 `기본형 모바일 안내페이지`에 가까웠다면, 이번 6종은 양식 자체를 다르게 나눴다.

- 기본형: 정보 정리
- 룩북형: 이미지와 감성
- 포스터형: 주목도와 이벤트
- 피드형: 인스타 프로필 연결감
- 견적형: 상담 전 필터링
- 캘린더형: 가능 시간 우선
- 사례형: 결과 증명

다음 단계는 각 양식별로 관리자 수정 영역을 붙여서 `사진/가격/공지/예약상태`를 바꿔보는 데모를 만드는 것.
