모바일 안내페이지 양식 편집기

사진·가격표·이벤트 입력 → 양식 선택 → HTML 내보내기

사장님 입력 패널

바꾸는 즉시 오른쪽 미리보기에 반영
1. 기본 정보상호/위치/문의
2. 첫 화면 문구고객이 바로 보는 제목
3. 품목/가격대표 3개
4. 이벤트/혜택포스터형에서 크게 노출
5. 이미지 URL대표/보조/B&A
최종 판매 방향

랜딩 대행이 아니라
입력형 모바일 안내페이지

크몽에서는 “사진과 가격표만 주시면 인스타 프로필용 모바일 안내페이지를 제작해드립니다”로 판매하고, 이후 자체 사이트에서는 9,900원 자동 생성 상품으로 확장할 수 있는 구조입니다.

1 입력상호, 가격, 이벤트, 사진 URL만 받습니다.
2 양식 선택목적에 따라 룩북/포스터/인스타/예약/사례 구조가 바뀝니다.
3 내보내기편집기 없는 고객용 HTML을 다운로드합니다.

현재 입력 데이터

19,000원~29,000원. 완성형 랜딩페이지가 아닌 저가형 안내페이지 양식으로 포지셔닝.
차별점
색상 변경이 아니라 전환 목적별 정보 구조를 바꾸는 편집기.
