# 사비 원칙 기반 5가지 모바일 랜딩 양식 샘플

## 목적
결과물을 보고 이 작업이 가치 있는지 판단하기 위한 고객-facing 샘플 팩.

## 샘플
1. 룩북형 갤러리 — 이미지 선택/디자인 문의 중심
2. 이벤트 포스터형 — 혜택/가격/마감 중심
3. 인스타 프로필 허브형 — 링크인바이오/빠른 탐색 중심
4. 예약 캘린더형 — 가능한 날짜/예약 전 정보 중심
5. 비포애프터 사례형 — 문제/과정/결과/후기 중심

## 판단 포인트
색상만 다른 템플릿이 아니라, 업종과 목적에 따라 섹션 순서·정보 구조·CTA 방식이 달라질 수 있는지를 확인한다.
