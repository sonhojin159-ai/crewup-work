사진으로 오염도를 먼저 확인하고 기본 범위/추가 가능성을 사전 안내한 뒤 진행했습니다.

상황

진행 과정

작업은 이렇게 진행됐습니다

1. 사진 5장 확인

주방, 욕실, 창틀, 바닥 상태를 먼저 확인했습니다.

2. 기본/추가 범위 분리

후드 내부 깊은 분해는 추가 확인 대상으로 분리했습니다.

3. 작업 후 사진 공유

완료 후 같은 구도 사진을 전달해 확인 시간을 줄였습니다.
