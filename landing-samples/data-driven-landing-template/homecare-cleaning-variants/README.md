# 입주청소/홈케어 모바일 랜딩페이지 양식 5종

## 생성 목적
예약 학습 결과를 바탕으로 특정 업종을 하나 골라 실제 판매 가능한 모바일 홍보페이지 양식 5종을 제작했다.

## 선택 업종
입주청소/이사청소/홈케어 서비스

## 선택 이유
- Antigravity 학습에서 청소/이사/가전수리 홈케어 서비스는 구매 가능성이 높고 19,000~29,000원 양식 구매 적합성이 매우 높다고 정리됨.
- 고객이 확인하려는 항목이 명확함: 서비스 범위, 추가요금, 전후사진, 책임보험, A/S, 지역, 일정, 카톡 견적.
- Codex 학습에서 case/before-after, price-list, consultation/FAQ, event-poster, calendar/booking 패턴이 홈케어와 잘 맞음.

## 포함 양식
1. 01-instant-quote-calculator: 30초 견적 계산형
2. 02-before-after-proof: 전후사진 사례형
3. 03-area-schedule-checker: 지역·일정 확인형
4. 04-trust-faq-consultation: 신뢰·FAQ 상담형
5. 05-limited-event-poster: 이벤트 포스터형

## 공통 반영 학습
- 히어로 → 가격/상황 → 신뢰/주의사항 → 하단 고정 CTA 구조
- CTA는 카톡/전화 중심으로 2개 이내
- 현장 확인 후 견적 변동 가능성을 명시
- 전후사진에는 평수, 지역, 작업시간 같은 메타데이터를 붙임
- 고객 입력 필드는 디자인 옵션보다 사진, 가격, 이벤트, 지역, 일정, CTA 링크, 주의사항 우선
