# PT/체형교정 문의 전환 샘플 랜딩페이지

## 선택 업종

`1:1 체형교정 PT / 자세교정 / 통증관리형 퍼스널트레이닝`

## 추천 이유

- 상담/예약 전환이 명확하다.
- 가격, 후기, 진행 방식, 위치, FAQ가 고객 판단에 직접 영향을 준다.
- 카카오톡 문의 CTA와 잘 맞는다.
- 사진/후기/전후 변화로 신뢰 요소를 만들기 쉽다.
- 제작 후 공지, 가격, 시간표, 후기 업데이트가 자주 발생해 월관리 전환 명분이 있다.

## 설계 방향

- 모바일 퍼스트 1페이지
- 히어로에서 즉시 `상담 가능`, `지역`, `카카오톡 문의` 노출
- 서비스/가격/FAQ/진행 방식을 카드형으로 정리
- 하단 스티키 CTA로 문의 버튼 상시 노출
- 대표가 수정 가능한 필드 데모 포함
