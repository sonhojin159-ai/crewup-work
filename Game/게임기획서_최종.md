## 1. 게임 개요

용사 팀과 마왕 팀이 시련을 사이에 두고 벌이는 심리전 보드게임.
마왕은 시련을 배치하고, 용사는 동료와 용기를 선택하여 맞서 싸운다.
**양쪽 모두 팀원 간 상의 불가** — 각자의 자리에서 각자의 상황에 최선을 다해야 이기는 게임.
이전 라운드 결과만을 보고 다음 수를 읽는 머리싸움이 핵심.

## 2. 구성 요소

### 2-1. 시련 카드 (마왕 사용)

| 시련 | 설명 |
|------|------|
| 시련 A | (테마 미정) |
| 시련 B | (테마 미정) |
| 시련 C | (테마 미정) |
| 시련 D | (테마 미정) |

- 총 8장 (4종 x 2장씩)
- 게임 전체에서 각 종류 최대 2회 사용 가능

### 2-2. 용기 카드 (용사 사용)

| 용기 | 대응 시련 |
|------|----------|
| 용기 a | 시련 A |
| 용기 b | 시련 B |
| 용기 c | 시련 C |
| 용기 d | 시련 D |

- 4종류, 매 라운드 시련 수만큼 선택
- 동일 용기를 중복 선택 가능

### 2-3. 동료 (용사 팀 사용)

**특정 동료 (4종):**

| 동료 | 능력 |
|------|------|
| 동료 A | 해당 라운드에서 시련 A 등장 시 자동 승리 |
| 동료 B | 해당 라운드에서 시련 B 등장 시 자동 승리 |
| 동료 C | 해당 라운드에서 시련 C 등장 시 자동 승리 |
| 동료 D | 해당 라운드에서 시련 D 등장 시 자동 승리 |

- 매 라운드 해당 용사가 1명 선택
- **팀 내 중복 불가**: 이전 라운드에서 팀원이 사용한 동료는 재선택 불가
- 패시브 발동: 해당 시련 등장 시 자동으로 정화 포인트 획득
- 발동 시 마왕 팀에게 동료 정체가 공개됨

**만능 동료 (1종):**

| 동료 | 능력 |
|------|------|
| 만능 동료 | 아래 모드별 특수 능력 참조 |

- 만능 동료 선택 시 게임 전체에서 특정 동료 사용 불가 (팀 전원 적용)
- 정체가 절대 공개되지 않음 (마왕 팀은 어떤 동료인지 추론만 가능)

## 3. 게임 모드

### 3-1. 공통 규칙

- 총 6번의 시련, 매 시련마다 정화 또는 타락 포인트가 1점씩 쌓임
- 시련과 용기가 일치 → **정화** +1
- 시련과 용기가 불일치 → **타락** +1
- 동료가 자동 승리한 시련 → **정화** +1 (용기 선택과 무관)
- 게임 종료 시 **정화 >= 타락이면 용사 팀 승리**, 타락 > 정화이면 마왕 팀 승리
- 양쪽 모두 팀원 간 **상의 불가**
- 라운드 배정은 게임 시작 시 **랜덤**
- 각 라운드의 용사와 마왕은 **동시에 선택** (시련 확인 전에 용기/동료 선택)
- 2라운드 이후: 양쪽 모두 **이전 라운드 결과를 확인한 후** 자기 라운드 수행

### 3-2. 2 vs 2 (2라운드 x 시련 3개)

```
라운드 구조:  R1 [시련 3개]  →  R2 [시련 3개]
배정 (랜덤): 용사A vs 마왕A → R1  /  용사B vs 마왕B → R2
```

| 라운드 | 마왕 | 용사 |
|--------|------|------|
| R1 | 마왕A: 8장에서 시련 3개 배치 (블라인드) | 용사A: 동료 1명 + 용기 3개 선택 |
| R2 | 마왕B: R1 결과 확인 후 잔여 카드에서 시련 3개 배치 | 용사B: R1 결과 확인 후 동료 1명 + 용기 3개 선택 |

- 양쪽 모두 팀원과 **상의 불가**
- R2의 용사B와 마왕B는 R1 결과를 보고 판단 → 정보 우위
- 라운드 배정이 랜덤 → 누가 블라인드(R1)를 맡을지 모름

**특정 동료:** 라운드마다 해당 용사가 1명 선택 (팀 내 중복 불가)
**만능 동료:**
- R1: 용사A가 시련 슬롯 1개를 지정하여 무조건 승리 + 용기 2개
- R2: 마왕B가 잔여 카드풀(5장)에서 시련 3장 먼저 선택 → 용사B가 1장 밴 → 밴 카드 공개 → 마왕 선택에 밴 카드 포함 시 마왕B 재선택 + 용기 3개

### 3-3. 3 vs 3 (3라운드 x 시련 2개)

```
라운드 구조:  R1 [시련 2개]  →  R2 [시련 2개]  →  R3 [시련 2개]
배정 (랜덤): 용사A vs 마왕A → R1  /  용사B vs 마왕B → R2  /  용사C vs 마왕C → R3
```

| 라운드 | 마왕 | 용사 |
|--------|------|------|
| R1 | 마왕A: 8장에서 시련 2개 배치 (블라인드) | 용사A: 동료 1명 + 용기 2개 선택 |
| R2 | 마왕B: R1 결과 확인 후 잔여 카드에서 시련 2개 배치 | 용사B: R1 결과 확인 후 동료 1명 + 용기 2개 선택 |
| R3 | 마왕C: R1+R2 결과 확인 후 잔여 카드에서 시련 2개 배치 | 용사C: R1+R2 결과 확인 후 동료 1명 + 용기 2개 선택 |

- 양쪽 모두 팀원과 **상의 불가**
- 정보량: R1(블라인드) < R2(R1 확인) < R3(R1+R2 확인) — 양쪽 동일
- 카드 선택지: R1(8장) > R2(6장) > R3(4장) — 마왕 측
- 정보는 많아지지만 선택지는 줄어드는 구조 → 라운드마다 다른 긴장감
- 라운드 배정이 랜덤 → 전략적으로 중요한 R3를 누가 맡을지 모름

**특정 동료:** 라운드마다 해당 용사가 1명 선택 (팀 내 중복 불가, 4종 중 3종 사용)
**만능 동료:**
- R1: 용사A가 시련 슬롯 1개를 지정하여 무조건 승리 + 용기 1개
- R2~R3 중 **1회만** 밴 사용 가능:
  - 밴 사용 라운드: 마왕이 시련 먼저 선택 → 해당 용사가 **2장 밴** → 밴 카드 포함 시 마왕 재선택 + 용기 2개
  - 밴 미사용 라운드: 용기 2개만으로 대응 (동료 능력 없음)
- 밴 사용 여부는 해당 라운드 용사가 독립적으로 결정 (팀원과 상의 불가)
- 단, 밴은 팀 전체에서 1회만 가능 → R2 용사가 사용하면 R3 용사는 사용 불가

## 4. 라운드 진행 순서

### 특정 동료 선택 시

```
1. [마왕] 시련 카드 배치 (잔여 카드풀에서 선택, 뒷면으로)
2. [용사] 동료 1명 선택 + 용기 카드 선택 (시련 수만큼)
   - 팀원이 사용한 동료는 선택 불가
3. [동시 공개] 시련 카드와 용기 카드를 모두 공개
4. [판정]
   - 동료의 대응 시련이 존재 → 해당 시련 자동 정화 (동료 정체 공개)
   - 나머지 시련: 용기 카드와 매칭 확인
   - 일치 → 정화 +1 / 불일치 → 타락 +1
5. [기록] 사용된 시련 카드를 카드풀에서 제거
6. [공개] 양쪽 모두 결과 확인 → 다음 라운드 담당자가 이를 보고 판단
```

### 만능 동료 선택 시

```
1. [R1] 용사A가 자동 승리 슬롯 번호를 선택
2. [R1] 마왕A가 시련 배치 → 동시 공개
3. [R1 판정] 지정 슬롯 = 무조건 정화 / 나머지 = 용기 매칭 판정
4. [R2 이후 — 밴 사용 라운드]
   a. 마왕이 잔여 카드풀에서 시련 카드를 먼저 선택 (비공개)
   b. 해당 용사가 잔여 카드풀에서 밴 선택 → 밴 카드 공개
      (2v2: 1장 밴 / 3v3: 2장 밴)
   c. 마왕의 선택에 밴 카드가 포함되어 있으면:
      → 해당 카드 제거 → 마왕이 잔여 풀(밴 제외)에서 재선택
   d. 용사 용기 선택 → 동시 공개 → 용기 매칭 판정
5. [R2 이후 — 밴 미사용 라운드]
   a. 마왕이 잔여 카드풀에서 시련 배치
   b. 용사 용기 선택 → 동시 공개 → 용기 매칭 판정
```

## 5. 동료 정체 공개 규칙

| 상황 | 공개 여부 |
|------|----------|
| 특정 동료가 발동됨 (대응 시련 등장) | **공개** — 마왕 팀이 동료 종류를 알게 됨 |
| 특정 동료가 발동 안 됨 (대응 시련 미등장) | **비공개** — 마왕 팀은 추론만 가능 |
| 만능 동료 (R1 자동 승리) | **비공개** — 어떤 슬롯이 자동 승리인지만 공개, 동료 종류는 비공개 |
| 만능 동료 (R2+ 밴) | **비공개** — 밴된 카드는 공개되지만, 동료 종류 추론 불가. 마왕은 카드 선택 후 밴 결과를 확인하며, 밴 적중 시 재선택 |

**핵심:** 마왕 팀은 동료 카드가 발동되기 전까지 용사의 동료를 알 수 없다.
특정 동료 발동 → 정체 노출 → 다음 라운드 마왕이 해당 시련 회피 가능 (단, 팀 내 중복 불가로 같은 동료 재사용 불가).
만능 동료 → 정체가 절대 노출되지 않음 → 마왕 팀은 끝까지 읽기 싸움.

## 7. 보상 시스템

### 판돈 설정

- 매 게임(모험) 시작 전, 참가자들이 판돈을 합의하여 설정
- 최소/최대 판돈은 게임 내 재화 시스템에 따라 결정

### 최종 포인트 계산

```
넷 정화 = 정화 포인트 - 타락 포인트  (용사 팀 승리 시)
넷 타락 = 타락 포인트 - 정화 포인트  (마왕 팀 승리 시)
```

### 보상 분배

**용사 팀 승리 (정화 >= 타락):**

```
각 마왕이 지불: 판돈 x (넷 정화 + 1)
각 용사가 수령: 판돈 x (넷 정화 + 1)
```

- +1 보너스: 용사 도전에 대한 보상 (최소 승리인 3:3에서도 이득)
- 팀 규모가 같으므로 1:1 대응 — 각 마왕의 손실 = 각 용사의 수익

**마왕 팀 승리 (타락 > 정화):**

```
각 용사가 지불: 판돈 x 넷 타락
각 마왕이 수령: 판돈 x 넷 타락
```

### 보상 예시 (판돈 100)

**용사 팀 승리 — 정화 5, 타락 1 (넷 정화 4, 대승):**

| 모드 | 각 마왕 손실 | 각 용사 수익 |
|------|------------|------------|
| 2v2 | -500 | **+500** |
| 3v3 | -500 | **+500** |

**용사 팀 승리 — 정화 3, 타락 3 (넷 정화 0, 간신히 승리):**

| 모드 | 각 마왕 손실 | 각 용사 수익 |
|------|------------|------------|
| 2v2 | -100 | **+100** |
| 3v3 | -100 | **+100** |

**마왕 팀 승리 — 정화 1, 타락 5 (넷 타락 4, 대패):**

| 모드 | 각 마왕 수익 | 각 용사 손실 |
|------|------------|------------|
| 2v2 | +400 | **-400** |
| 3v3 | +400 | **-400** |

### 보상 구조 특징

- 팀 규모 대칭 → 개인별 보상은 모드 무관하게 동일
- 넷 포인트 방식 → 대승/대패의 보상 차이가 극적
- +1 보너스 → 50% 근처 승률에서 용사 기대값 약간 양수 → 도전 동기 부여
- 개인의 라운드 성적이 팀 전체 결과에 직결 → 책임감 + 긴장감

## 8. 밸런스 데이터

### 용사 팀 승률 (이론값, 최적 혼합전략 기준)

| 모드 | 특정 동료 | 만능 동료 |
|------|----------|----------|
| 2v2 | 45~50% | 45~55% |
| 3v3 | 47~51% | 47~53% |

> **참고:** 양쪽 모두 팀원 간 상의 불가이므로, 팀 내 조율 부재가 양쪽에 동일하게 적용 → 대칭 구조로 밸런스가 자연스럽게 수렴.
> 특정 동료는 팀 내 중복 불가로 후반 라운드 동료 후보가 좁아짐 → 마왕 읽기 확률 상승.
> 만능 동료는 밴 적중 시 강력하나 미적중 시 효과 없음 → 분산 증가.
> 3v3 만능 동료는 R2~R3 중 1회만 밴(2장), 밴 사용 여부를 해당 용사가 독립 판단 (조율 불가).

### 전략별 특성

| 동료 선택 | 강점 | 약점 | 플레이 스타일 |
|----------|------|------|-------------|
| 특정 동료 | 매 라운드 자동 승리 기회 | 발동 시 정체 노출 + 팀 내 중복 불가로 후반 읽힘 | 심리전, 읽기 싸움 |
| 만능 동료 | 정체 비공개 + 2장 밴 적중 시 마왕 강제 배정 | R1 자동 승리 1회 고정 + 밴 1회뿐(미사용 라운드 무방비) | 정보전, 도박형 |

### 모드별 특성

| 모드 | 참가 인원 | 전략 깊이 | 게임 시간 | 추천 |
|------|----------|----------|----------|------|
| 2v2 | 4명 | 높음 (대칭 2라운드 읽기) | 짧음 | 입문용/캐주얼 |
| 3v3 | 6명 | 최고 (3라운드 정보 누적 + 팀 조율 불가) | 중간 | 하드코어 |

## 9. 핵심 전략 포인트

### 용사 팀 전략

1. **동료 선택의 딜레마:** 동료가 발동하면 포인트 확보, 하지만 정체 노출 + 팀 내 중복 불가로 다음 용사의 선택지가 좁아짐
2. **팀원 행동 추론:** 상의 불가이므로 "팀원이 어떤 동료를 썼을까"를 결과에서 추론해야 함. 발동 안 된 동료는 정체 비공개 → 다음 용사도 확신 불가
3. **만능 동료의 밴 타이밍:** 3v3에서 밴은 1회뿐. R2 용사가 쓰면 R3 용사는 무방비. 하지만 상의 불가이므로 "상대가 안 썼겠지"라는 읽기가 필요

### 마왕 팀 전략

1. **시련 분산 vs 집중:** 분산하면 동료 적중 확률 상승, 집중하면 용기 매칭 확률 하락
2. **정보 활용:** 이전 라운드에서 발동된 동료를 제외하면 다음 용사의 동료 후보가 좁아짐 → 적극 활용
3. **밴 대비 (만능 동료 상대 시):** 밴 당할 가능성을 고려하여 시련 분산. 핵심 시련 1종에 집중 시 밴 적중하면 치명적
4. **카드풀 관리:** 후반 라운드로 갈수록 선택지 축소. 앞 마왕의 카드 소모를 보고 남은 풀에서 최적 배치

### 팀 간 대칭 역학

- 양쪽 모두 상의 불가 → "팀원이 뭘 할지" 읽기가 양쪽에 동일하게 적용
- R1 담당(양쪽): 블라인드, 카드풀 최대 → 자유롭지만 정보 없음
- R2+ 담당(양쪽): 이전 결과 확인 가능, 하지만 카드풀 축소(마왕) / 동료 풀 축소(용사)
- 라운드 배정이 랜덤 → 전략적으로 중요한 후반 라운드를 누가 맡을지 운명에 달림
- "같은 조건에서 누가 더 잘 읽는가"가 순수하게 승부를 가름

## 11. 밸런스 참고 사항

- 대칭 구조(양쪽 상의 불가 + 동일 정보량)로 자연스럽게 50% 수렴
- +1 보상 보너스로 용사 팀의 기대 수익이 약간 양수 → 도전 인센티브
- 3v3 만능 동료의 밴은 R2~R3 중 1회(2장), 해당 용사가 독립 판단 → 팀 조율 불가가 밸런스 역할
- 라운드 배정 랜덤 → 운의 요소가 팀 구성에 개입하여 리플레이성 향상
- 시련/용기 테마에 따라 직관적인 매칭 관계를 부여하면 초보자 접근성 향상

## 12. 캐릭터 고유 능력 (확장 시스템)

> 기본 규칙 위에 얹어지는 확장 시스템. 프로토타입 완성 후 순차 구현 예정.
> 모든 능력은 동급 밸런스 원칙 — 레어도는 능력 강도가 아닌 보상 배율에만 차등.
> 남/여 캐릭터 버전 분리 → 일러스트 수집 요소 2배.

### 12-1. 용사 캐릭터 능력 (7종)

| # | 이름 | 능력 | 트레이드오프 |
|---|------|------|-------------|
| 1 | 정찰 | 마왕이 배치한 시련 중 1장을 미리 확인 (슬롯은 랜덤) | 정보 우위, 하지만 1장뿐 |
| 2 | 밴 | 라운드 시작 전 마왕 카드풀에서 시련 1장 제거 | 마왕 선택지 축소, 하지만 뭘 밴할지는 읽기 싸움 |
| 3 | 교체 | 시련 공개 후 자신의 용기 카드 1장을 다른 용기로 교체 | 사후 대응 가능, 하지만 1장 한정 |
| 4 | 추가 선택 | 용기 카드를 시련 수 +1장 선택 (먼저 매칭되는 것 적용) | 매칭 확률 상승, 하지만 분산 리스크 |
| 5 | 거울 | 마왕이 이번 라운드에서 가장 많이 쓴 시련 종류 1개를 자동 정화 | 몰빵 시 강력, 분산 배치 시 약함 |
| 6 | 복사 | 이전 라운드 팀원이 사용한 동료 능력을 그대로 복사 | R2 이후에만 유효, R1 배정 시 효과 없음 |
| 7 | 타락 용사 | 승리 조건 반전: 타락 > 정화이면 용사 팀 승리 | 팀 내부 읽기 싸움 유발, 양날의 검 (에픽 한정) |

### 12-2. 동료 능력 (6종)

| # | 이름 | 능력 | 트레이드오프 |
|---|------|------|-------------|
| 1 | 방어 | 특정 시련(A/B/C/D) 등장 시 자동 정화 | 시련이 안 나오면 무효, 발동 시 정체 공개 |
| 2 | 용기 증폭 | 해당 라운드 용기 카드 +1장 추가 선택 | 매칭 확률 상승, 하지만 자동 정화 없음 |
| 3 | 예지 | 마왕 시련 중 1장의 종류를 미리 확인 | 용기 선택에 도움, 나머지 시련은 모름 |
| 4 | 역전 | 불일치 1개를 정화로 뒤집음, 대신 일치 1개도 타락으로 뒤집힘 | 순수 도박형. 불일치 많으면 이득, 전부 일치면 손해 |
| 5 | 연쇄 | 이번 라운드 정화가 2개 이상이면 추가 정화 +1 보너스 | 고위험 고보상. 정화 0~1이면 완전 무효 |
| 6 | 흡수 | 타락 1개를 정화로 전환, 단 동료 자동 방어 능력 포기 | 확실한 1포인트 전환, 방어형 능력은 못 씀 |

### 12-3. 마왕 캐릭터 능력 (9종)

| # | 이름 | 능력 | 트레이드오프 |
|---|------|------|-------------|
| 1 | 예언 | 용사의 동료를 예측 선언 → 맞추면 동료 무력화 | 맞추면 초강력, 틀리면 완전 무효 |
| 2 | 과부하 | 시련 1종을 3회까지 사용 가능 (기본 2회 제한 해제) | 몰빵 전략 가능, 읽히면 역이용당함 |
| 3 | 위장 | 시련 1장의 뒷면에 가짜 표시 가능 (공개 시 진짜 드러남) | 용사 추론 교란, 공개 후에는 효과 없음 |
| 4 | 봉인 | 라운드 시작 전 용사의 용기 1종을 사용 금지 선언 | 용사 선택지 축소, 뭘 봉인할지도 읽기 싸움 |
| 5 | 추가 배치 | 시련을 1장 더 배치 (용사 용기 수는 그대로 → 추가 시련은 자동 타락) | 확정 타락 1점, 카드풀 1장 추가 소모 |
| 6 | 정보 차단 | 이번 라운드 사용한 시련 종류를 다음 라운드에 비공개 | 상대 정보 차단, 자기 팀 마왕도 못 봄 |
| 7 | 인큐버스 | 여성 용사/동료의 능력 무효화 | 상대가 여성이면 강력, 남성이면 완전 무효 |
| 8 | 서큐버스 | 남성 용사/동료의 능력 무효화 | 상대가 남성이면 강력, 여성이면 완전 무효 |
| 9 | 증폭 | 이번 라운드 모든 포인트 2배 (정화도 2배, 타락도 2배) | 양날의 검 — 해당 라운드가 승부의 빅 라운드가 됨 |

### 12-4. 시스템 규칙

- **성별 속성:** 모든 용사/동료는 남성 또는 여성 버전으로 존재 (같은 능력, 다른 일러스트)
- **레어도:** 일반 / 레어 / 에픽 — 능력 강도 동일, 보상 배율만 차등 (하이리스크 하이리턴)
- **뽑기(가챠) 3개 라인:** 용사 뽑기 / 마왕 뽑기 / 동료 뽑기
- **과금 원칙:** 일러스트/스킨 판매 전용. 능력 강화 없음 → 순수 실력 대전 유지
- **보상 해금:** 게임 보상으로 일러스트 해금 (능력 강화 아님)
- **타락 용사:** 에픽 등급 한정, 팀 내 1명 제한 권장
