## 홈 인기 크루 섹션 변경 요약
홈 탭에는 전체 목록이 아니라 큐레이션 성격의 인기 크루 일부만 보이도록 분리했습니다.

- 섹션명: `현재 인기 크루`
- 큰 제목: `지금 주목받는 크루`
- 설명: `최근 신청과 활동성이 높은 크루를 먼저 보여드립니다.`
- 노출 카드 수: 전체 목록 중 일부 카드만 홈에 노출
- CTA: `전체 크루 목록 보기`
  - 클릭 시 `크루 목록` 탭으로 이동

## 전체 크루 목록 탭 신설/분리 방식
기존에는 홈 안의 크루 섹션으로 `크루 목록` 메뉴가 스크롤되는 구조였으나, 별도 탭으로 분리했습니다.

- 새 탭 컨테이너: `tab-content-crews`
- 섹션 제목: `전체 크루 목록`
- 카테고리 필터 위치: 전체 크루 목록 탭 내부
- 필터 항목:
  - 전체
  - 콘텐츠 제작
  - 부업
  - 웹페이지 제작
  - 자동화
  - AI 학습/스터디

## 내비게이션/CTA 동작 변경 요약
- 상단 `크루 목록` 메뉴: 홈 내부 스크롤이 아니라 `tab-content-crews` 탭으로 이동
- Hero `크루 목록 보기`: `크루 목록` 탭으로 이동
- 홈 인기 크루 섹션의 `전체 크루 목록 보기`: `크루 목록` 탭으로 이동
- 마지막 CTA `나에게 맞는 크루 찾기`: `크루 목록` 탭으로 이동
- 모바일 하단 `크루 목록 보기`: `크루 목록` 탭으로 이동

## 더미데이터/placeholder 처리 방식
홈 인기 크루와 전체 크루 목록의 카드는 모두 시안용 placeholder입니다.
실제 구현 시 크루명, 크루장명, 모집 수, 완주율, 인기 배지 등은 DB/API/CMS/관리자 입력 데이터로 대체해야 합니다.
관련 원칙은 `/workspace/MA/crewup_dummy_data_guardrail.md`를 따릅니다.
